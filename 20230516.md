cv2.destroyAllWindows()

### 잠만보 대칭

```python
import cv2

src = cv2.imread("snor.jpeg", cv2.IMREAD_COLOR)
dst = cv2.flip(src, 0)

cv2.imshow("src", src)
cv2.imshow("dst", dst)
cv2.waitKey()
cv2.destroyAllWindows()
```
